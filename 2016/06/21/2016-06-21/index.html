Mac 下反编译 apk

2016-06-21
Android 反编译 brew
  1. 1，环境准备
  2. 2，反编译 代码
  3. 3，反编译资源文件
  4. 4，重新打包
  5. 5，签名(需要 java 环境变量)
  6. 6，对齐操作（程序运行的更快）

1，环境准备

brew install apktool
brew install dex2jar
jd-gui

2，反编译 代码

解压 apk 拿到 dex 文件
执行 d2j-dex2jar classes.dex 获取到 jar 文件，拖进 jd-gui 就可以了

3，反编译资源文件

执行 apktool d *.apk
-f 如果目标文件存在，删除存在
-o 指定目标文件夹名字
-s 不反编译 dex 文件
-r 不反编译资源文件

4，重新打包

apktool b *.apk -o new.apk

5，签名(需要 java 环境变量)

jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore 签名文件 -storepass 签名密码 待签名 apk 签名的别名

6，对齐操作（程序运行的更快）

zipalign 4 *.apk *_aligned.apk
